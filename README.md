## Appium BDD Test Automation

This is a mobile demo test automation project. Test Automation is done on Qatar Airways application. It covers a plane ticket purchase scenario wholly.

### Tech Stack

* Appium        7.3.0
* Gauge         1.0.7
* Gauge-Java    0.7.3
